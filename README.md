A Minimal and Elegant Framework for Real-Time Interactive World Models.

Turning any bidirectional video model into a real-time interactive world model with action-conditioned interaction.

## Features
- Fully open-source end-to-end pipeline
- Support both HunyuanVideo and Wan
- Real-time action-conditioned interaction

## Pipeline
1. Data pipeline  
2. Bidirectional action-conditioned finetuning  
3. Autoregressive finetuning  
4. AR video diffusion distillation  
5. Real-time inference  

Currently organizing training/inference framework, demos, and open-source docs.  
Expected release in ~2 days 🚀
